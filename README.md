# Nix Dofiles

My nix configuration for 
- MacOS
- Arch Linux (work in progress)


## Usage

### MacOS
- rebuild and switch system configuration
```
darwin-rebuild switch --flake ~/nix#mac
```

### Arch Linux
- *work in progress*
```
home-manager switch --flake ~/nix#arch
```
